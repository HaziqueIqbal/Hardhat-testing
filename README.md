# Hardhat-testing
Testing some basic functions of zombie factory using hardhat and waffle.
## Dependencies
**Node JS**
## Installation
> npm install --save-dev hardhat
## Start
> npx hardhat
## Guide
Open hardhat.config.js file and write **require("@nomiclabs/hardhat-waffle");**
## Compile
> npx hardhat compile
## Test
> npx hardhat test
